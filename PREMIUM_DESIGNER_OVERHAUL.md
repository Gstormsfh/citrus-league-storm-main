## üéØ Design System Applied

### Typography Hierarchy
```
H1/H2:    font-varsity (Graduate) - 5xl-7xl, uppercase, black weight
H3/H4:    font-display (Montserrat) - bold, uppercase
Body:     font-sans (Inter) - medium weight, relaxed leading
Labels:   font-display - semibold, uppercase, tracked
```

### Color Usage
```
Backgrounds:  citrus-cream (#FFFDF2)
Text:         citrus-forest (#1B3022)
Primary:      citrus-sage (#AAD1A3)
Secondary:    citrus-peach (#EFCCC6)
Accent:       citrus-orange (#DF7536)
Muted:        citrus-charcoal (#333333)
```

### Border System
```
Standard:     2px minimum
Emphasis:     3px for important elements
Heavy:        4px for varsity patches
Radius:       rounded-2xl (1rem) or rounded-varsity (2rem)
```

### Shadow System
```
shadow-patch:   0 4px 0 0 rgba(27, 48, 34, 0.1)
shadow-varsity: 0 6px 0 0 rgba(27, 48, 34, 0.15), 0 2px 8px rgba(0, 0, 0, 0.1)
```

### Animation System
```
Hover Lift:     -translate-y-1 or -translate-y-2
Button Press:   translate-y-0.5 or translate-y-1
Duration:       transition-all duration-300
Easing:         ease-out (default)
```
